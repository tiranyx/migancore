**Impact:** Tidak blocking — API tetap berfungsi normal.

**Temuan:** Field `commit_sha` di health response menunjukkan `"unknown"`.
**Penyebab:** Docker build tidak inject `COMMIT_SHA` sebagai build arg.
**Action:** Low priority — tambahkan `--build-arg COMMIT_SHA=$(git rev-parse HEAD)` di deploy script.
