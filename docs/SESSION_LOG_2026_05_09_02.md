## 3. TEMUAN & LESSON LEARNED

### 🔍 Temuan Teknis

#### Temuan 1: Test hafidz_not_in_base_metadata_yet Fail Setelah Model Import
Setelah menambahkan `HafidzContribution` ke `models/__init__.py`, test `test_hafidz_not_in_base_metadata_yet` fail karena tabel sekarang terdaftar di `Base.metadata`. 

**Fix:** Update assertion dari `not in` ke `in` — test sekarang memverifikasi bahwa model properly registered.

#### Temuan 2: Integration Test Setup Butuh Model Registration
`Base.metadata.create_all()` hanya membuat tabel untuk model yang sudah di-import. Tanpa `from models.hafidz import HafidzContribution` di `models/__init__.py`, integration test gagal karena tabel `hafidz_contributions` tidak ada.

**Lesson:** Always register new models in `models/__init__.py` agar `Base.metadata` aware.

#### Temuan 3: Git Bundle Push via VPS Berfungsi Konsisten
Pattern `git bundle → scp → ssh git pull bundle HEAD → git push origin main` sudah teruji 3× dan bekerja reliable.
