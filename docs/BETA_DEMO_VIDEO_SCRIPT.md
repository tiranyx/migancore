## 🎤 NO VOICEOVER (per riset Indonesian DM context)

- WhatsApp video = silent autoplay
- User scroll cepat, voice often muted
- Visual storytelling > narration
- Kalau add caption text: max 5 kata di bawah, font besar putih

---

## 📱 RECORDING TIPS (Loom/native screen recorder)

### Setup before record:
1. Login dulu di `app.migancore.com` (jangan masuk login flow di video, friction)
2. Buka tab baru (clean state)
3. Resolusi browser: full-screen, hide bookmarks bar, hide other tabs
4. Browser zoom: 100% (default)
5. **JANGAN ada notification popup, OS clock, taskbar visible** — distraksi

### During record:
1. Quick mouse movement (jangan hover lama)
2. Tunggu image full render (jangan stop sebelum chat bubble selesai)
3. Don't pause/edit — record one take, redo kalau ada error

### Post-process (Loom default OK):
- Trim awal/akhir kalau ada delay
- TIDAK perlu music, watermark, effect — clean is better

## 🚨 ANTI-PATTERN — JANGAN

❌ Voiceover yang panjang ("Halo gw Fahmi, lagi build product baru...")
❌ Music/sound effect (distraksi)
❌ Slow-mo / dramatic effect (looks try-hard)
❌ Tutorial-style narration ("Ini cara untuk...")
❌ GIF (Indonesian WhatsApp convention = meme = casual = trust hit)
❌ Show login screen (friction, kill conversion)
❌ Show error/loading state (no negative first impression)

✅ DO:
✅ Pure screen recording, action only
✅ One specific WOW moment (image generation in 5 sec)
✅ Crisp, clean, fast
✅ End immediately after WOW (don't show next steps)

## 📊 KENAPA INI WORKS (research backing)

Per Indie Hackers AI launch retrospectives Q1 2025:

- **No voiceover** = WhatsApp scroll culture
