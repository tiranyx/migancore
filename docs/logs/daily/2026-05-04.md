### Lessons (Day 29)

1. **FastAPI route order matters** — `/{param}` catches more specific paths if declared first. Always put concrete paths before parameterized ones, OR use lifespan order checks. Added inline comment as anti-regression doc.

2. **Ollama CPU bottleneck is structural** — synthetic + distillation cannot run concurrently without one starving. Adapt: serialize them, prefer synthetic (faster pipeline, no API costs).

3. **Container startup slow with synthetic running** — fastembed model loading takes 60-90s when CPU is busy with synthetic. Plan deploys with longer wait windows.
